# Projeto-Final-Fabricio-Santos-T-cnicas-de-Desenvolvimento-de-Algoritmos
Gerenciador de cadastro desenvolvido em PHP e MYSQL. Funcionalidades são cadastros de médicos e pacientes com editar e excluir. Utiliza Bootstrap 5 para o front-end e consultas SQL otimizadas. Perfeito para demonstrar conceitos de CRUD e modelagem de dados.
